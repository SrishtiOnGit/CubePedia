# CubePedia

-A webapp that will help you learn more about cubing instead of just solving. Basically, It's for beginners of cube. It will take you beyond just cube solving.

# Features
1.Champion Archieve:
Discover the people who changed speedcubing forever.
2.Cube Hardware:
Know about it's materials like lubes, magnets and more.
3.Popular Cubes:
Discover the most popular Rubik's Cubes.
4.Simple To Use:
It's simple to use and easy to navigate.
